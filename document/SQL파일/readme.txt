* 3, 4번 과정은 DML입니다. DML 실행시 문제가 발생할 경우 rollback을 해야하기 때문에 일부러 commit을 포함시키지 않았으니, 모든 내용을 진행한 뒤에는 반드시 commit을 해주셔야합니다.
